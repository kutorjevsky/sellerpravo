# Сервис справок осмотра веб-страниц

Минимальный веб-сервис, который формирует справку о состоянии веб-страницы на момент обращения.

## Что делает

- Принимает URL (`http/https`).
- Выполняет HTTP-запрос к странице.
- Фиксирует технические параметры:
  - дата/время осмотра (UTC),
  - исходный и конечный URL,
  - HTTP-статус,
  - `Content-Type`,
  - размер ответа,
  - SHA-256 хэш тела,
  - заголовок страницы,
  - найденные ссылки,
  - HTTP-заголовки ответа.
- Сохраняет справку в `reports/<report_id>.json`.
- Отдаёт справку в JSON и текстовом формате.

## Запуск

```bash
python3 web_inspection_service.py --host 127.0.0.1 --port 8080
```

После запуска откройте:

- `http://127.0.0.1:8080/` — веб-интерфейс;
- `POST http://127.0.0.1:8080/api/inspect` — API.

## Пример API-запроса

```bash
curl -sS -X POST http://127.0.0.1:8080/api/inspect \
  -H 'Content-Type: application/json' \
  -d '{"url":"https://example.com"}'
```

В ответе будут ссылки на:

- `.../reports/<id>.json`
- `.../reports/<id>.txt`

## Важно

Сервис формирует **техническую справку осмотра** и не подменяет нотариальное удостоверение.
